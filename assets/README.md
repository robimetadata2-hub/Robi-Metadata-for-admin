# Asset Folder Structure

This folder is used to store static assets like images and logos for the application.

Please create the following directory structure inside your project's root:

```
/
├── assets/
│   ├── images/
│   │   ├── stock-logos/
│   │   │   ├── adobe-stock.png
│   │   │   ├── dreamstime.png
│   │   │   ├── freepik.png
│   │   │   ├── general.png
│   │   │   ├── shutterstock.png
│   │   │   └── vecteezy.png
│   │   ├── logo.png
│   │   └── profile.png
... (other project files)
```

## Instructions

1.  **Create the folders:** Make sure the `assets`, `assets/images`, and `assets/images/stock-logos` folders exist in your project.
2.  **Add your main logo:** Place your company logo inside the `assets/images/` folder and name it `logo.png`.
3.  **Add the profile picture:** Place the profile picture for the login modal inside `assets/images/` and name it `profile.png`.
4.  **Add stock site logos:** Place the corresponding logos for each stock photography site inside the `assets/images/stock-logos/` folder. The filenames should match the ones listed above.

This will resolve any broken image links in the application.
